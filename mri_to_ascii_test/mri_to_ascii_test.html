mri_to_ascii_test

mri_to_ascii_test, a C program which calls mri_to_ascii() to read a file containing MRI information in binary format, and writes an equivalent ASCII text file.

Related Data and Programs:

mri_to_ascii, a C program which converts the binary data in a magnetic resonance imaging (MRI) file to ASCII text.

Source Code:

  • mri_to_ascii_test.sh, the source code;

  • roi.mri, the header part of the MRI file;
  • roi.dat, the main part of the MRI file;
  • roi.txt, the ASCII text translation of roi.dat, created by the command mri_to_ascii roi.
