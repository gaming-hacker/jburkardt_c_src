trig_to_neib_test

trig_to_neib_test, a C program which calls trig_to_neib() to read files defining the nodes and elements of a triangulation, and compute a list of the node neighbors, and, optionally, information about the Voronoi decomposition of the region.

trig_to_neib, a C program which reads NODE and ELE files (a format prescribed by triangle) describing a triangulation, and produces a file defining the neighbor triangles of each element, and information about the Voronoi diagram, by Lili Ju.

ELL is a set of example data which corresponds to a mesh using 21 nodes and 24 elements, constituting a mesh of an L-shaped domain.

  • ell.node the node file.
  • ell.ele the element file.
  • ell.neib the node neighbor file.
  • ell.voro the Voronoi file.

GREENLAND is a set of example data which corresponds to a mesh using 33,343 nodes and 64,125 elements, constituting a mesh of Greenland.

  • greenland.node the node file.
  • greenland.ele the element file.
  • greenland.neib the node neighbor file.
  • greenland.voro the Voronoi file.
