cc_io_test

cc_io_test, a C program which calls cc_io() to read and write compressed column (CC) sparse matrix files.

Related Data and Programs:

cc_io, a C library which reads and writes sparse linear systems stored in the Compressed Column (CC) format.

Source Code:

  • cc_io_test.c a sample calling program.
  • cc_io_test.sh runs all the tests.
  • cc_io_test.txt, the output file.

Test #1 and #2 write and read back the information for the "simple" 5x5 matrix with 12 nonzero entries. There are files associated with this.

  • simple_acc.txt, the values.
  • simple_ccc.txt, the (0-based) compressed column indices.
  • simple_icc.txt, the (0-based) row indices.
