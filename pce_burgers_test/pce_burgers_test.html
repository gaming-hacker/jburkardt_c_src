pce_burgers_test

pce_burgers_test, a C program which calls pce_burgers() to define and solve a version of the time-dependent viscous Burgers equation, with uncertain viscosity, using a polynomial chaos expansion, in terms of Hermite polynomials.

pce_burgers, a C program which defines and solves the time-dependent viscous Burgers equation, with uncertain viscosity, using a polynomial chaos expansion in terms of Hermite polynomials, by Gianluca Iaccarino.

Source Code:

  • pce_burgers_test.sh, runs all the tests.

  • burgers.history.txt, the value of the solution (expansion coefficients) at selected times.
  • burgers.modes.txt, the modes of the solution at the final time.
  • burgers.moments.txt, the mean and variance of the coefficients at each point, at the final time.
