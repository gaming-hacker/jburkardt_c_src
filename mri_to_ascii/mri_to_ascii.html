MRI_TO_ASCII
Conversion of MRI data to Text Format

MRI_TO_ASCII, a C program which reads a file containing MRI information in binary format, and writes an equivalent ASCII text file.

Usually, the ASCII data is created so that it can be modified in some way; if it is necessary to convert the modified data back to the MRI format, the ascii_to_mri command can be used.

Usage:

mri_to_ascii prefix
where
  • prefix.dat is the name of the binary file containing the MRI information.
  • prefix.txt is the ASCII text file containing a readable copy of the MRI information.
