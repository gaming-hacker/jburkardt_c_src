A is a set of data defining the outline of a capital letter A. Files you may copy include:

  • A.poly, a sample POLY data file;
  • A.1.poly, the new POLY file created by TRIANGLE;
  • A.1.node, the node file created by TRIANGLE;
  • A.1.ele; the element file created by TRIANGLE;
  • A.1.ele.png; A PNG image of the elements;

square_circle_hole is a set of data defining a square region, with a circular hole, filled up with CVT points. Files you may copy include:

  • square_circle_hole.poly, a sample POLY data file;
  • square_circle_hole.1.poly, the new POLY file created by TRIANGLE;
  • square_circle_hole.1.poly.png, a PNG image of the region and points;
  • square_circle_hole.1.node, the node file created by TRIANGLE;
  • square_circle_hole.1.ele; the element file created by TRIANGLE;
  • square_circle_hole.1.ele.png, a PNG image of the elements;
