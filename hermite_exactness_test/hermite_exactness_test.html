hermite_exactness_test

hermite_exactness_test, a C program which calls hermite_exactness() to investigate the polynomial exactness of a Gauss-Hermite quadrature rule for the infinite interval (-oo,+oo).

Related Data and Programs:

hermite_exactness, a C program which tests the monomial exactness of quadrature rules for estimating hermite-type integrals of a function f(x) with density rho(x)=exp(-x^2) over the interval (-oo,+oo).
