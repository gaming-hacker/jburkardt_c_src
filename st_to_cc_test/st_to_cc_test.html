st_to_cc_test

st_to_cc_test, a C program which calls st_to_cc() to convert information describing a sparse matrix from sparse triplet (ST) format to compressed column storage (CC).

st_to_cc, a C library which converts sparse matrix data from Sparse Triplet (ST) format to Compressed Column (CC) format.
