asa136_test

asa136_test, a C program which calls asa136() to divide M points in N dimensions into K clusters so that the within-clusters sum of squares is minimized.

Related Data and Programs:

asa136, a C library which implements a clustering algorithm, by Hartigan and Wong. This is a version of Applied Statistics Algorithm 136.

Source Code:

  • asa136_test.c, calls all the tests.
  • asa136_test.sh, runs all the tests.
  • asa136_test.txt, the output file.

  • asa136_test_points.txt, a sample dataset of 100 points in 2D, to be arranged into 5 classes.
