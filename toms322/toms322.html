TOMS322
The Fisher Test

TOMS322, a C library which computes Fisher's F-ratio test.

The original C source code for this program, as written by Mark Johnson, is available at http://www.netlib.org/opt/hooke.c

The text of many ACM TOMS algorithms is available online through ACM: http://calgo.acm.org/ or NETLIB: http://www.netlib.org/toms/index.html.
