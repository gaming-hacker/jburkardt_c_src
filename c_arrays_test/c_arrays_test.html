arrays_test is a very simple example of how to set up arrays in C. In particular, we note that vectors are easy to declare with fixed or dynamic dimensions, but that arrays of dimension 2 or greater aren't easy to declare dynamically. The example shows how to work around by settting up a two dimensional array as a one dimensional vector.

  • arrays_test.c, the source code;
  • arrays_test.sh, runs all the tests;
  • arrays_test.txt, the output file;

pointers_test shows how a two-dimensional array can be defined, either as row or column major, using pointers. Unfortunately, the column major format means that the double indexing lists the column index first!

  • pointers_test.c, the source code;
  • pointers_test.sh, runs all the tests;
  • pointers_test.txt, the output file;
