JACOBI
The Jacobi Iteration for symmetric positive definite (SPD) Linear Systems

JACOBI, a C library which sets up the Jacobi iteration for a symmetric positive definite (SPD) linear system.

The program uses the GNUPLOT_I interface program to create plots of the residual and the "motion" of the solution over the iteration.
